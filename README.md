## &#128204; About Project

**native-neo-student-form** adalah pure frontend project untuk website **Student Registration Form / Formulir Pendaftaran Mahasiswa Baru**.

Project ini dibuat dengan pendekatan **Neo Brutalism UI**: warna terang, thick black borders, hard offset shadows, bold typography, playful layout, hover lift animation, dan button press effect.

**Tech rules:**

- &#9989; HTML
- &#9989; CSS
- &#9989; Vanilla JavaScript only
- &#10060; No React
- &#10060; No framework
- &#10060; No external libraries untuk website

---

## &#128640; Features

- &#9889; Spectacular loading screen sebelum halaman tampil
- &#128202; Animated progress bar
- &#129000; Bouncing colorful shapes
- &#128172; Dynamic loading text
- &#10024; Smooth transition dari loading screen ke main page
- &#129504; Form validation dengan JavaScript
- &#128680; Required field error message
- &#127881; Success popup modal setelah submit berhasil
- &#128260; Reset button
- &#129521; Brutalist cards dengan thick border dan hard shadow
- &#128070; Hover lift animation dan button press effect
- &#128241; Responsive layout untuk mobile dan desktop
- &#128450;&#65039; Clean separated files: HTML, CSS, dan JS

---

## &#128221; Form Sections

- &#128100; **Data Pribadi**: nama, NIK, tempat lahir, tanggal lahir, jenis kelamin, email, WhatsApp, alamat
- &#127979; **Data Pendidikan**: asal sekolah, jurusan asal, tahun lulus, nilai rata-rata
- &#127919; **Program Studi**: pilihan program studi dengan card-style radio buttons
- &#128104;&#8205;&#128105;&#8205;&#128103; **Data Orang Tua / Wali**: nama wali, hubungan, pekerjaan, nomor HP
- &#128206; **Berkas Lampiran**: checklist dokumen pendaftaran
- &#9989; **Persetujuan**: final confirmation sebelum submit
