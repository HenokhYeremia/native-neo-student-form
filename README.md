# About Project

Website formulir pendaftaran mahasiswa baru dengan tema **Neo Brutalism Colorful UI** menggunakan:

- HTML
- CSS
- Vanilla JavaScript

Tanpa framework.  
Tanpa React.  
Tanpa library tambahan.

Pure frontend project.

---

# Features

- Spectacular Loading Screen
- Neo Brutalism Design
- Colorful Modern UI
- Responsive Mobile Layout
- Form Validation
- Success Popup Modal
- Hover Animation
- Button Press Effect
- Brutalist Shadow Style
- Interactive JavaScript Effects
